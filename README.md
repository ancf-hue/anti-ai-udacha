# 🎲 Проект «Анти ИИ — Удача»

> **ИИ рассчитывает. Человек рискует.**

## 🔍 Что это?

Мини-приложение, которое проверяет, насколько твои решения непредсказуемы для искусственного интеллекта. 
Никакой логики, только спонтанный выбор. 10 попыток — 3 карты — 1 результат.

## ✨ Особенности

- 📱 Работает **полностью офлайн** — без интернета, без серверов
- 🔒 **Приватность**: не собирает данные, не отслеживает, не отправляет ничего в сеть
- 🎙️ Озвучка результатов голосом ведущего
- 🎉 Визуальные эффекты: конфетти, салют, анимации при победе
- ♿ Доступность: поддержка `prefers-reduced-motion`
- 🌐 Адаптивный дизайн для мобильных и десктопов
